/join

With this command you join a team to participate in the match. Players with a purchased rank can also use "/join (team)" to join a specific team.


/leave

Leave the match and get put on the observer team.


/match

Displays match information in chat.


/toggle

Toggle a setting. Use "/toggle (setting)" to toggle that setting.


/settings

List of settings that can be used with the /toggle command.


/report

Report a player to alert staff if they are breaking the rules. Read reporting for more information.


/staff

Check for online staff.


/g & /t

Switch between global and team chat. Running the command before your message will say it in that chat. If you don't add any extra message it will switch the primary chat.


/msg

Send a private message to a player. You can also use /r to quickly reply to a player that messages you last.


/friends

Checks for online friends. You can add friends through the website.


/server

Used for switching servers and displays the current server you are on. To switch servers use "/server (server name)".


/servers

Lists all servers.


/map

Provides more information about the map you are playing. You can also use "/map (map name)" to check for that specific map. The map must be in the Overcast maps repository.


/rot

Lists the servers map rotation.


/next

Tells the next map.


/seen

The command checks for when a player was last online. Do "/seen (player name)".
