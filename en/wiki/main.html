History

Overcast Network started with Dewtroid streaming and creating the first DTC map (Airship Battle) on stream. People then started submitting maps and the OCN-style map making scene was created. This inspired Apple and Anxuiz to make PGM. At first it was used exclusively to host community days because that is the only way it could get enough players but then became a stand alone server which is now known as the Overcast Network.
